
 Folders in desktop/:

--------------------------------
gdm/
 + Gramado Desktop Manager.
   This application is used to handle the access to a new session 
   in a given desktop.

--------------------------------
setup/
 + Setup application
   This application is used to install the operating system.
   If the desktop was initialized but the operating system was 
   no intalled yet, so we use this application to properly install the 
   operating system into a selected partition and setup the bootloader.

---------------------------------
taskbar/
 + This is the taskbar.
   This application is used as a launcher for the client-side 
   GUI applications.
