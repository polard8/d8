# rtl

OS base interface for ring3 applications. 

## Environment

```
    + Ring 3.
    + userland/ subsystem.
```

## What we can find here?

```
    + Syscall interrupts for x86_64 archtecture.
    + Syscalls to request kernel services.
    + Internal libraries.
    + Posix-like C library.     
```
