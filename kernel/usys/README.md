# userland - Programs running in ring 3.

> * bin/ - The place for the final binaries.

> * core00/ - Libraries and toolkits.
> * core01/ - Init systems.
> * core02/ - Shells.
> * core03/ - Tests.

> * ftpd/ - FTP server.
> * netd/ - Network server.
> * telnet/ - Telnet server. 
> * vgad/ - VGA service.
> * webd/ - Web server.
